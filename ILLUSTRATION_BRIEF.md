# Tarot Card Illustration Brief

20 pre-generated images to be placed in `public/illustrations/`. The app maps each image to a card via filename — no code changes needed once the files are in place.

---

## Technical Specifications

| Property | Value |
|---|---|
| **Format** | PNG |
| **Dimensions** | 810 × 1029 px |
| **Aspect ratio** | ~4:5 portrait (270:343 native display size × 3) |
| **Background** | Opaque — no transparency needed |
| **Text** | None — no labels, numbers, or lettering |
| **Borders** | None — the app draws card frames separately |
| **Delivery path** | `public/illustrations/{id}.png` |

---

## Shared Style Prompt

Prepend this to every image prompt for visual consistency across the set:

> tarot card illustration, art nouveau style, rich jewel tones, intricate linework, symbolic composition, dramatic chiaroscuro lighting, centered subject, no text, no borders, no card frame, painterly

## Generation Tips

- Run all 20 through the same model and style seed if possible so they feel like a cohesive set
- For Midjourney use `--ar 4:5`
- The art nouveau + jewel-tone style works across both light cards (The Sun, The Star) and dark ones (The Moon, The Tower, The Devil) due to its inherent contrast
- Drop finished PNGs into `public/illustrations/` — the app picks them up automatically with no code changes
